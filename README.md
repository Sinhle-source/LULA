This project is a BDD framework that applied the Page Object Model. Our Page Object Model approach is a Page FactoryTo execute and generate a response, please follow the below steps:

Extract the zipped project.
Import the project to your local machine/work environment.(IntelliJ IDE is recommended).
The feature file contains three scenario blocks.
Click the "TestRunner" package to see the test runner class. The class is already in order and don't change anything.
Right Click the test runner class to run the project.
After execution is completed, navigate to "target/cucumber-reports/cucumber.html" for test report.
